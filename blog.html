An explanation for this Blog

It contains two types: Links, represented in <\ aside> elements in the <\ main>, and Blog Posts represented as <\ article> elements. Initially, all the posts from the last month that a post was made should appear on the page. THe <\ a.load-more> link below should link to the next month with articles (as a permalink). When loading this permalink it will fetch the contents of that <\ main> elements and append it to the page using a script. If no scripts are available, it will just link to that months permalink. The URL will be auto updated to allow for this. Every post has a permalink of the type blog/year/month/slug. So this one would be blog/2020/september/an-explanation-for-this-blog.
